# آیکون‌های افزونه

برای افزونه Chrome، به سه اندازه آیکون نیاز دارید:

- `icon16.png` - 16x16 پیکسل
- `icon48.png` - 48x48 پیکسل  
- `icon128.png` - 128x128 پیکسل

## ساخت آیکون

می‌توانید از ابزارهای آنلاین مثل:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- یا هر ابزار طراحی دیگر

یا از SVG زیر استفاده کنید و آن را به PNG تبدیل کنید:

```svg
<svg width="128" height="128" viewBox="0 0 128 128" fill="none" xmlns="http://www.w3.org/2000/svg">
  <rect width="128" height="128" rx="24" fill="#6366f1"/>
  <path d="M64 32L32 48L64 64L96 48L64 32Z" stroke="white" stroke-width="4" stroke-linecap="round" stroke-linejoin="round"/>
  <path d="M32 80L64 96L96 80" stroke="white" stroke-width="4" stroke-linecap="round" stroke-linejoin="round"/>
  <path d="M32 64L64 80L96 64" stroke="white" stroke-width="4" stroke-linecap="round" stroke-linejoin="round"/>
</svg>
```

# محاسبه هزینه Cutup برای یک ویدئو 1 دقیقه‌ای

## 📊 هزینه‌های API

### 1. Whisper API (Transcription)
- **مدل**: `whisper-1`
- **قیمت**: **$0.006 per minute**
- **برای 1 دقیقه**: **$0.006**

### 2. GPT-4o-mini (Summarization)
- **مدل**: `gpt-4o-mini`
- **قیمت Input**: $0.15 per 1M tokens
- **قیمت Output**: $0.60 per 1M tokens

**تخمین برای 1 دقیقه ویدئو:**
- متن تولید شده: ~150-200 کلمه (≈ 200-300 tokens)
- Prompt + متن: ~400-500 tokens input
- خلاصه + نکات کلیدی: ~150-200 tokens output

**محاسبه:**
- Input: (500 / 1,000,000) × $0.15 = **$0.000075**
- Output: (200 / 1,000,000) × $0.60 = **$0.00012**
- **مجموع Summarization**: **~$0.0002**

### 3. GPT-4o (Correction - اختیاری)
- **مدل**: `gpt-4o` (فقط برای متن فارسی)
- **قیمت Input**: $2.50 per 1M tokens
- **قیمت Output**: $10.00 per 1M tokens

**تخمین:**
- Input: ~500 tokens
- Output: ~300 tokens
- **هزینه**: ~$0.0015-0.002 (فقط برای فارسی)

### 4. Translate SRT (اختیاری)
- **مدل**: `gpt-4o-mini`
- **تخمین**: ~$0.0003-0.0005

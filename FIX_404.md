# رفع خطای 404

## 🔍 مشکل

خطای 404 یعنی endpoint پیدا نمی‌شود. این می‌تواند به دلایل زیر باشد:

1. **URL در popup.js قدیمی است** - deployment جدید URL جدید دارد
2. **Deployment جدید انجام نشده** - function deploy نشده
3. **Route درست کار نمی‌کند** - vercel.json مشکل دارد

## ✅ راه‌حل

### 1. گرفتن URL جدید

در Vercel Dashboard:
1. به **Deployments** بروید
2. روی deployment **Current** کلیک کنید
3. URL را کپی کنید (مثلاً `https://cutup-xxxxx.vercel.app`)

### 2. آپدیت popup.js

در `popup.js`، خط 2 را آپدیت کنید:

```javascript
const API_BASE_URL = 'URL_NEW_DEPLOYMENT'; // URL جدید را اینجا بگذارید
```

### 3. Deploy جدید (اگر لازم است)

در Terminal:

```bash
cd "E:\Machine Learning\cutup"
vercel --prod
```

بعد از deploy، URL جدید را از Vercel Dashboard بگیرید و در `popup.js` آپدیت کنید.

## 📋 چک‌لیست

- [ ] URL جدید از Vercel Dashboard گرفته شده
- [ ] `popup.js` آپدیت شده
- [ ] Extension reload شده
- [ ] تست انجام شده

## 🎯 بعد از رفع مشکل

وقتی URL را آپدیت کردید:
1. Extension را reload کنید
2. یک فایل صوتی کوچک تست کنید
3. باید لاگ `=== TRANSCRIBE FUNCTION CALLED V4.0 ===` را ببینید
